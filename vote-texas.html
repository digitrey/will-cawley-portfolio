Introduction

This redesign focused on enabling Texas voters to easily access state level resources to help them vote in general, midterm, and off cycle elections.

Website

VoteTexas.gov is a website that helps Texans exercise their right to vote. It is run by the Office of The Secretary of State, who manages elections and voting in Texas.

VoteTexas helps users check their registration status, learn how to register, and know what they need to do on Election Day.

The Problem

Despite these invaluable functions, VoteTexas is not very user friendly or visually impressive. The mobile version is responsive in on some pages and difficult to navigate in others.

Because voting is so important, any website that help voters should be thoroughly tested and tweaked to better accommodate all people. This redesign seeks to refine the website to improve its use.
